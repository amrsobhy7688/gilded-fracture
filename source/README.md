## Google Doc Shortcuts

`google-doc-shortcuts/` contains the `.gdoc` shortcut files for the related Drive docs, preserving their document IDs and titles. Some related Docs require authenticated Google access for export, so their shortcuts are archived here instead of fake HTML login exports.
